About

System Prompt Open open-sources system prompts extracted by JustAsk, a self-evolving framework that autonomously discovers effective extraction strategies through interaction alone. No handcrafted prompts, no labeled supervision, no privileged access — just standard user interaction.

JustAsk formulates extraction as an online exploration problem, using Upper Confidence Bound–based strategy selection and a hierarchical skill space. These skills exploit imperfect system-instruction generalization and inherent tensions between helpfulness and safety.

In March 2026, Claude Code's source was leaked via a .map file in the npm registry. We compared it section-by-section against our JustAsk extractions from January 2026 — two months before the leak — achieving 85–95% verified accuracy.
